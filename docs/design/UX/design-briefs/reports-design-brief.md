# 1. Overview

## Purpose

The Reports module provides comprehensive business insights by transforming inventory data into meaningful visual analytics. It enables users to monitor inventory performance, identify trends, evaluate stock movements, and export reports for operational and strategic decision-making.

This screen emphasizes clarity, readability, and actionable insights while maintaining consistency with the official Pharmora Design System.

---

# 2. Business Goals

The Reports screen aims to:

- Provide a centralized analytics dashboard.
- Improve inventory visibility through data visualization.
- Support operational and strategic decision-making.
- Monitor inventory performance over time.
- Simplify report generation and exporting.
- Reduce manual reporting processes.

---

# 3. Target Users

Primary Users

- Pharmacy Administrator
- Pharmacy Owner
- Inventory Manager
- Store Manager

Secondary Users

- Finance Staff
- Operations Team

# 4. Design Goals

The Reports screen should communicate information clearly while remaining visually calm and professional.

# 6. Screen Components

## Header

Displays:

- Page Title
- Description
- Date Range Selector
- Export Button

---

## Statistics Cards

Display four KPI cards.

Cards:

- Total Inventory Value
- Inventory Turnover
- Stock Movements
- Low Stock Rate

Each card includes:

- Icon
- Main Metric
- Trend Indicator
- Supporting Caption

---

## Inventory Trend

Primary analytics section.

Visualization:

- Line Chart

Metrics:

- Incoming Stock
- Outgoing Stock
- Inventory Balance

Time Period:

- Weekly
- Monthly
- Quarterly

---

## Category Distribution

Visualization:

Horizontal Bar Chart

Displays:

- Medicines
- Medical Devices
- Supplements
- Consumables
- Others

Each category includes percentage contribution.

---

## Inventory Movement

Visualization:

Stacked Bar Chart

Metrics:

- Stock In
- Stock Out
- Adjustments

Display monthly comparisons.

---

## Recent Reports

Displays recently generated reports.

Columns:

- Report Name
- Report Type
- Generated By
- Created Date
- Status
- Actions

Actions:

- View
- Download
- Export

---

## AI Insights

Displays intelligent business recommendations.

Example insights:

- Fast-moving inventory
- Slow-moving inventory
- Overstock risk
- Reorder recommendation
- Category performance

Display:

- Confidence Score
- Recommendation
- Primary Action Button

# 11. Empty State

If no reports exist:

Display:

- Empty Illustration
- Empty Title
- Description
- Generate Report Button

Example:

"No reports have been generated yet."

---

# 12. Loading State

Display:

- Skeleton KPI Cards
- Skeleton Charts
- Skeleton Table

Avoid layout shifting.

---

# 13. Error State

Display:

- Error Illustration
- Error Description
- Retry Button

Example:

"Unable to load report analytics."

---

# 14. Success State

Display toast notifications.

Examples:

- Report exported successfully.
- Report generated successfully.
- Report downloaded successfully.

# 19. Design Constraints

The Reports screen must:

- Follow the official Pharmora Design System.
- Use the official spacing scale.
- Use the official typography.
- Reuse existing button styles.
- Reuse existing table styles.
- Reuse existing cards.
- Reuse existing sidebar and navigation.
- Maintain visual consistency with Dashboard, Products, Categories, Suppliers, and Inventory.

Do not introduce:

- Glassmorphism
- Heavy gradients
- Excessive colors
- Decorative illustrations
- Unnecessary widgets

# Password Reset Success — Design Brief

## Overview

The Password Reset Success page confirms that the user's password has been successfully updated.

This page serves as the final step in the password recovery flow and reassures users that their account is now secure.

The design should communicate trust, success, and confidence while remaining visually consistent with the Pharmora authentication experience.

---

# Objective

Clearly communicate that the password has been changed successfully and guide users back to the Login page.

---

# Target Users

- Pharmacy Manager

This page should feel like the final step of the authentication journey.

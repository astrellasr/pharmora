# Reset Password — Design Brief

## Overview

The Reset Password page allows users to securely create a new password after accessing a valid password reset link.

This page should focus on clarity, security, and simplicity while maintaining the premium enterprise appearance of the Pharmora authentication experience.

The layout must remain visually consistent with the Login, Register, and Forgot Password pages.

---

# Objective

Allow users to securely set a new password with a frictionless experience.

---

# Target Users

- Inventory Administrator
- Pharmacy Manager
- Clinic Staff
- Healthcare Organization Admin

---

# Layout

Two-column authentication layout.

## Left Panel

Same as:

- Login

- Forgot Password

DO NOT redesign this section.
